# P5_Quiz
Aplicación web de servidor generada por express-generator.
Basada en la estructura MVC:
* Modelo (model)
* Vista (view)
* Controlador (controller)
Las vistas cuentan con un molde común (layout.ejs) en el que sólo cambia el cuerpo de la página.
